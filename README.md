1. PARCEIRO FEZ INSCRIÇÃO SORTEIO
   OUTRO PARCEIRO FEZ INSCRIÇÃO SELECIONANDO ESSE PARCEIRO
   PARCEIRO CONFIRMOU PARCEIRO 
   PRONTO

2. PARCEIRO FEZ INSCRIÇÃO COM OUTRO PARCEIRO
   OUTRO PARCEIRO FEZ INSCRIÇÃO COM O PARCEIRO
   AMBOS CONFIRMARAM
   PRONTO

3. PARCEIRO E O OUTRO PARCEIRO FIZERAM INSCRIÇÃO EM SORTEIO
   PARCEIRO SELECIONOU O OUTRO PARCEIRO EM COMPRAS
   OUTRO PARCEIRO CLICOU NO BOTÃO DE SORTEIO QUE APARECE QUANDO NÃO TEM PARCEIRO SELECIONADO EM COMPRAS
   APARECE UMA TELA DE CONFIRMAR PARCEIROS
   OUTRO PARCEIRO CONFIRMA PARCEIRO
   PRONTO

4. PARCEIRO E O OUTRO PARCEIRO ESTÃO CONFIRMADOS
   PARCEIRO SELECIONOU OUTRO PARCEIRO2 EM COMPRAS
   A VENDA DO OUTRO PARCEIRO FOI PARA SORTEIO
   OUTRO PARCEIRO2 CONFIRMOU INSCRIÇÃO COM O PARCEIRO
   PRONTO
